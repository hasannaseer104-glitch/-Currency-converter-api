# Currency Converter

A simple, real-time currency converter web application that allows users to convert amounts between different currencies using live exchange rates. It dynamically updates currency flags and fetches data from a free API.

## 🚀 Features
- **Live Exchange Rates**: Fetches real-time currency conversion rates using the `@fawazahmed0/currency-api`.
- **Dynamic Flag Icons**: Automatically displays the national flag of each selected currency using `flagsapi.com`.
- **Dropdown Selection**: Pre-filled with over 150 global currencies for easy selection.
- **Input Validation**: Automatically sets the minimum amount to `1` if left empty or invalid.
- **Instant Conversion**: Calculates and displays the converted amount with a single click.

## 🛠️ Tech Stack
- **HTML5** – Structure and layout.
- **CSS3** – Styling, responsive design, and flexbox layout.
- **JavaScript (ES6+)** – DOM manipulation, API fetching, and dynamic logic.
- **REST API** – External currency data API.
